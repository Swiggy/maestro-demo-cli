# Contributing

Thanks for your interest in contributing to `maestro-demo-cli`!

## Getting started

```bash
git clone https://github.com/Swiggy/maestro-demo-cli.git
cd maestro-demo-cli
yarn install
yarn build
```

## Development

```bash
yarn dev        # watch mode (tsc --watch)
yarn test       # run tests
yarn test:watch # run tests in watch mode
```

## Submitting changes

1. Fork the repo and create a branch from `master`
2. Make your changes and add tests if applicable
3. Ensure `yarn test` passes
4. Open a pull request with a clear description of what you changed and why

## Reporting bugs

Open an issue with:
- What you ran
- What you expected to happen
- What actually happened
- Your Node version (`node --version`) and OS
